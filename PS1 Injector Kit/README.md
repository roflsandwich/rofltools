# ps_shellcode

Usage:
```
- Create .bin file
- Run bin2ps.py on .bin file and pipe output to another file. eg: .\bin2ps.py payload.bin > a
- Host file on webserver
- Edit injector.ps1 and point to webserver
- Run injector.ps1
```

*V2*
Usage: 
```
- Point merged_v2 to bin file
- Run merged_v2.ps1
```
